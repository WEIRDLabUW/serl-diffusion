## Installation
1. Conda Environment:
    create an environment with
    ```bash
    conda create -n serl python=3.10
    ```
2.  [serl_franka_controller](https://github.com/rail-berkeley/serl_franka_controller)
3. pip install the `serl_robot_infra` folder
